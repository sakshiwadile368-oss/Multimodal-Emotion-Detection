# 🎭 Multimodal Emotion Detection

## 📖 Project Overview

This project detects human emotions using three different modalities:

- 😊 Face Emotion Detection
- 💬 Text Emotion Detection
- 🎤 Speech Emotion Detection

The outputs from these three models are combined using a fusion technique (majority voting) to predict the final emotion.

---

## ✨ Features

- Upload an image for face emotion detection
- Enter text for text emotion detection
- Upload an audio file for speech emotion detection
- Fuse all three predictions
- Display the final detected emotion
- Provide emotion-based recommendations
